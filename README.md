# LED Bilbordi

Професионален уебсайт за наем на LED билборди в България.

## Технологии

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Hook Form + Zod
- Lucide React (икони)

## Инсталация

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Отворете [http://localhost:3000](http://localhost:3000) в браузъра.

## Build

```bash
npm run build
```

## Форматиране

```bash
npm run format
```
